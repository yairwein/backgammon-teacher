# Backgammon Teacher

A teaching-focused backgammon web application built with SvelteKit. Play against the computer, get your moves analyzed by GNU Backgammon, and receive human-readable explanations of your mistakes.

## Features

- **Play vs Computer** — Play a full game with move-by-move analysis
- **Position Review** — Load positions, choose moves, and get detailed analysis
- **Blunder Detection** — Configurable equity threshold for identifying mistakes
- **Feature Extraction** — Structured position features (pip counts, blots, primes, anchors, etc.)
- **LLM Explanations** — Natural language coaching powered by Claude or GPT-4
- **Interactive Board** — Click-to-move interface with legal move highlighting

## Architecture

```
src/lib/
  backgammon/    Board representation, move generation, rules
  analysis/      GNU Backgammon adapter and output parsing
  features/      Position feature extraction
  llm/           LLM explanation layer (Anthropic/OpenAI/mock)
  game/          Game state management, computer player, blunder classification
  components/    Svelte UI components (Board, AnalysisPanel, Dice)
  data/          Sample positions
  db/            Database client

src/routes/
  /              Home page
  /play          Play vs computer mode
  /review        Position review mode
  /settings      User settings
  api/analyze    Position analysis endpoint
  api/computer-move  Computer move endpoint
  api/explain    LLM explanation endpoint
  api/game       Game management endpoint
```

## Prerequisites

- Node.js 18+
- GNU Backgammon (`gnubg`) installed and available in PATH
- (Optional) Anthropic or OpenAI API key for LLM explanations

### Installing GNU Backgammon

**macOS:**
```bash
brew install gnubg
```

**Ubuntu/Debian:**
```bash
sudo apt-get install gnubg
```

## Setup

```bash
# Install dependencies
npm install

# Generate Prisma client
npx prisma generate

# Push database schema
npx prisma db push

# Start dev server
npm run dev
```

## Environment Variables

Create a `.env` file:

```
DATABASE_URL="file:./dev.db"
GNU_BG_PATH="gnubg"
LLM_PROVIDER="mock"        # mock | anthropic | openai
LLM_API_KEY=""              # Required for anthropic/openai
```

## Modes

### Mode 1 — Teaching Review
1. Load or navigate through positions
2. GNU Backgammon analyzes each move
3. App identifies blunders according to user-defined threshold
4. App extracts structured explanatory features
5. LLM explains why the move is a blunder

### Mode 2 — Play vs Computer
1. Play a full game against the computer
2. Computer moves are generated by GNU Backgammon
3. Each user move is analyzed after it is played
4. If the move crosses the blunder threshold, the app shows an explanation
5. User can pause to review alternative moves

## Design Principles

1. **Engine truth** — GNU Backgammon is the authority on move strength
2. **Structured interpretation** — Feature extraction provides interpretable data
3. **Natural language teaching** — LLM explains using features, never invents facts
